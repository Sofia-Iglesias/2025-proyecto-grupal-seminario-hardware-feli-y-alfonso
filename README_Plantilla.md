## **`Objetivo`**

`el objetivo de este proyecto es hacer un riego automatico que riegue las plantas en base a la temperatura, la luz,
la humedad y un boton analogico`

## **`Descripción del Proyecto`**

`este proyecto consiste en un riego automatico y manual el cual esta compuesto por 4 entradas y 3 salidas. las entradas
serian sensores de temperatura, LDR, humedad y un boton. mientras que las salidas serian 2 leds uno verde para el boton
y uno azul para los sensores, un buzzer y un muultimetro. cuando se apreta el boton el buzzer y la luz quedan encendidos
durante 3 segundos`
